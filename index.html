Métricas do site

Acompanhe visitas, cliques e navegação pelo Google Analytics.

O site agora envia eventos personalizados para o GA4. Assim você consegue ver quais partes geram mais interesse e quais botões geram mais leads.

Eventos acompanhados automaticamente

Os eventos aparecem no Google Analytics em Relatórios > Engajamento > Eventos. Também dá para marcar o evento de WhatsApp como conversão.

site_loadedCarregamento e visita ao site.
whatsapp_clickClique em qualquer botão de WhatsApp.
section_navigationClique no menu e navegação por seções.
contact_form_submitEnvio do formulário de orçamento.
review_submitEnvio de avaliação/review.
page_viewEvento padrão do GA4 para visitas.

Como usar isso

Com esses eventos, você consegue entender o funil do site:

  • Quantas pessoas acessaram a página.
  • Quantas clicaram no WhatsApp.
  • Quais seções mais chamaram atenção.
  • Quantas enviaram formulário de orçamento.
  • Quantas deixaram review para aprovação.
Importante: os números globais ficam no GA4. O site dispara os eventos com segurança.

Dúvidas comuns

Antes de chamar, talvez isso já responda.

O projeto pode começar pequeno e evoluir. Muitas vezes, uma planilha bem feita ou um dashboard bem pensado já muda a rotina inteira.

Você faz só dashboard ou também planilha?

Faço os dois. Posso criar uma planilha personalizada do zero, melhorar uma existente ou transformar a planilha em dashboard e automação.

Preciso ter banco de dados para contratar?

Não. Posso trabalhar com Excel, Google Sheets, CSV, sistemas exportados, SQL, APIs ou montar uma estrutura nova conforme o projeto.

Você também faz sistema web?

Sim. Quando a solução passa do limite de uma planilha ou dashboard, posso criar aplicação com login, banco, API e área administrativa.
